# Exercice 2

Using the next operations, create expression and assing theirs values to variables.

a. == 
b. <=
c. >=
d. !=
e. <
f. >

## Solution GO

```go
package main

import "fmt"

func main() {
    a := 1 == 1
    b := 1 <= 1
    c := 1 >= 1
    d := 1 != 1
    e := 1 < 1
    f := 1 > 1
    fmt.Println(a, b, c, d, e, f)

    
}

```

## Result

=> true true true false false false
